# CIT Resewn (Legacy Fork for 1.21.x)

This is an unofficial **LEGACY fork** of the original [CIT Resewn](https://modrinth.com/mod/cit-resewn) mod, updated to work on **Minecraft 1.21.x** (specifically tested on 1.21.4).

I finally managed to port it!

---

## ‚ö†Ô∏è What is "LEGACY"?

This version is called **LEGACY** because it is a **stripped-down** port focusing **only on the core CIT (Custom Item Textures) functionality**.

To keep the mod lightweight and avoid unnecessary dependencies, the following have been **completely removed**:

*   **‚ùå No ModMenu integration**
*   **‚ùå No Cloth Config dependency**
*   **‚ùå No extra libraries**

‚úÖ **Pure CIT functionality, nothing else.**

This means there is no configuration screen, but the mod does exactly what it was made for: applying custom item textures from resource packs like Xalis Enchanted Books or any other OptiFine-compatible CIT packs.

---

## üì• Installation

1.  Ensure you have [Fabric Loader](https://fabricmc.net/) (version 0.16.0 or newer) and [Fabric API](https://modrinth.com/mod/fabric-api) installed for Minecraft 1.21.x.
2.  Download the `.jar` file from the [Releases](https://github.com/NRCreator/CITResewn-FORK-to-1.21.x/releases) section.
3.  Place the downloaded `.jar` file into your `mods` folder.
4.  Put your desired CIT resource packs into the `resourcepacks` folder and activate them in-game.

**For this mod to work, you need to install the file:**  
`citresewn-legacy-v0.9.1-1.21.4.jar`
