# CIT Resewn ‚Äî LEGACY/STANDALONE FORK for 1.21.x

This is an unofficial fork of the original [CIT Resewn](https://modrinth.com/mod/cit-resewn) mod, updated to work on **Minecraft 1.21.x** (tested on 1.21.4).

Two versions are available: **LEGACY** and **STANDALONE**.

---

## ‚öîÔ∏è LEGACY ‚Äî Stripped-down version

**LEGACY** is a **lightweight, stripped-down** port focusing **only on the core CIT (Custom Item Textures) functionality**.

To keep the mod minimal and dependency-free, the following have been **removed**:

*   ‚ùå **No ModMenu integration**
*   ‚ùå **No Cloth Config dependency**
*   ‚ùå **No extra libraries**

‚úÖ **Pure CIT functionality, nothing else.**

This version has no configuration screen ‚Äî it just works. Perfect if you only need custom item textures.

---

## üß© STANDALONE ‚Äî Full version

**STANDALONE** is the **complete, unmodified version**, based directly on the **original source code** with **no features removed**.

It includes:

*   ‚úÖ Full **ModMenu** integration
*   ‚úÖ Full **Cloth Config API** support
*   ‚úÖ All original settings and dependencies
*   ‚úÖ A full-featured configuration screen

**STANDALONE** means **"everything the original author intended"** ‚Äî all integrations, all settings, all dependencies. It is **self-contained** precisely *because* it keeps all these features.

## üì• Installation

1.  Ensure you have [Fabric Loader](https://fabricmc.net/) (version 0.16.0 or newer) and [Fabric API](https://modrinth.com/mod/fabric-api) installed for Minecraft 1.21.x.
2.  Choose your version:
    *   **LEGACY** ‚Äî for a lightweight, no-dependency experience.
    *   **STANDALONE** ‚Äî for the full-featured version with all integrations.
3.  Download the corresponding `.jar` file from the [Releases](https://github.com/NRCreator/CITResewn-FORK-to-1.21.x/releases) section.
4.  Place the downloaded `.jar` file into your `mods` folder.
5.  Put your desired CIT resource packs into the `resourcepacks` folder and activate them in-game.

**For the LEGACY version, install:**  
`citresewn-legacy-v0.9.1-1.21.4.jar`

**For the STANDALONE version, install:**  
*(coming soon ‚Äî will be named `citresewn-standalone-v0.9.1-1.21.x.jar`)*
